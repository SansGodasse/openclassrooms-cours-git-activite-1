Explications de git à mon fils

Dis papa, c'est quoi un commit ?

Alors un commit c'est en gros un genre de sauvegarde de ton projet à un moment donné.

Quand tu fais des modifications dans le repository de ton projet (ajout, modification, suppression de fichier par exemple), tu peux faire une sauvegarde de ton projet et y ajouter un descriptif des modifications effectuées. Cette sauvegarde s'appelle un commit.

Tu peux accéder à la liste des commits avec la commande git log. Tu y verras l'ensemble des commits de ton projet avec pour chacun d'eux un code unique appelé SHA qui te permettra d'y accéder avec la commande git checkout SHADuCommit, la date et l'heure de création du commit, l'auteur du commit et le message associé (si l'auteur s'est donné la peine d'en mettre un).

Je viens de te le dire ! La commande git log te permet d'afficher la liste de tous les commits d'un repository.
